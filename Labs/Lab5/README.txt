Lab 5 on Parallel computing course.
Generates histogram out of array of given size.
Usage:
	main [-s <arraySize>] [-m <histSize>] [-l] [-t] [-h]

	-s n	Set array size to n. Default 10000
	-m n	Set histogram size to n. Default 256
	-l	Run in linear mode. No threading.
	-t	Run tests.
	-h	Show this help.
